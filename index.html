Configurações de Publicação

Este gerador cria uma página HTML 100% autocontida da aba Visão Geral em tema claro (padrão de sites institucionais) contendo todos os dados, estilos e gráficos do Chart.js. Cole no WordPress ou hospede em seu servidor.

Código HTML Autocontido html
Como Publicar no WordPress:
  1. Clique em Copiar Código HTML acima.
  2. No painel do WordPress, edite a página onde deseja inserir o painel.
  3. Adicione um bloco de HTML Personalizado (Custom HTML) ou elemento HTML no Elementor/Divi.
  4. Cole o código copiado e salve a página. Pronto!
